## <Webアプリケーションの公開>

### Gradleを利用したWebアプリケーションの実行
以下のコマンドでホームディレクトリに移動します

    $ cd

cdコマンドを実行し,ディレクトリに移動します.

    $ cd trump

gradlewをbashを利用して実行します.

    $bash ./gradlew

プロジェクトの実行の為に以下のコマンドを実行します.

    $bash ./gradlew bootrun

これでWebアプリケーションが公開されるため, 以下のURLにアクセスできます.

URL:http://150.89.233.209
